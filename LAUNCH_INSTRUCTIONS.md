# Инструкция по запуску Sticker Processor Service

## ✅ Сервис успешно запущен!

### Что было сделано:

1. **Улучшен код конвертера** - добавлены множественные методы конвертации TGS в Lottie JSON:
   - Прямое распаковывание gzip (самый быстрый)
   - Библиотека Python `lottie`
   - CLI инструмент `tgs2json` (опционально)

2. **Исправлены deprecated предупреждения** - заменены `on_event` на современный `lifespan`

3. **Обновлена документация** - README теперь содержит актуальные инструкции

4. **Сервис запущен и работает** на порту 8081

### Доступные эндпоинты:

- **Health Check**: `http://localhost:8081/health`
- **Swagger UI**: `http://localhost:8081/docs`
- **Поддерживаемые форматы**: `http://localhost:8081/formats`
- **Получение стикера**: `http://localhost:8081/stickers/{file_id}`
- **Статистика кеша**: `http://localhost:8081/cache/stats`

### Как запустить сервис:

```bash
# Активировать виртуальное окружение
source venv/bin/activate

# Запустить сервис
uvicorn app.main:app --host 0.0.0.0 --port 8081 --reload
```

### Опционально - установка tgs2json для дополнительной поддержки:

```bash
npm install -g tgs2json
```

Сервис работает без этого инструмента, используя библиотеку `lottie` и прямое распаковывание gzip.
